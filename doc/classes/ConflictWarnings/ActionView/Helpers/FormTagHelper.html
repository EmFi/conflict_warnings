timestamp_tag(name = 'page_rendered_at')

Adds a timestamp with the given parameter name. Usage: 

    <%= timestamp_tag %>

produces 

   <input id=\"page_rendered_at\" name=\"page_rendered_at\" type=\"hidden\" value=\"#{Time.now.to_i}\" />
